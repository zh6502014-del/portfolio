⚠ 常見陷阱
這些地方容易改壞
陷阱說明正確做法
navLinks 快取nav-link 是非同步由 renderNav() 插入，不能在頁面載入時用 querySelectorAll 快取updateNav() 內每次動態查詢：document.querySelectorAll('.nav-link[data-section]')
model-viewer pointer-events.hero-3d 和 model-viewer 都有 pointer-events:none，無法直接監聽 touch/mouse 事件改用 document 監聽，getBoundingClientRect 判斷是否在容器內
Quill 段落在小網Quill 存的換行是 <p> 標籤，.hero-desc br { display:none } 無效renderProfile() 在 ≤560px 時把 <p> 合成一段 textContent
燈箱 touch 衝突3D 模型的 touchmove 用 passive:false，會吃掉燈箱的 touch 事件燈箱 touchstart 用 passive:true；e.target.closest('button') 時直接 return 不干涉
transform 順序scale() translate() 的 translate 單位會被 scale 放大，縮小後座標漂移改為 translate(x,y) scale(s)，pan 座標不受 scale 影響
Supabase SDK@supabase/supabase-js CDN 有 BroadcastChannel，觸發 DataCloneError已全部改用原生 fetch 打 REST API，不引入 SDK
glass-el 與自訂 background 衝突同時有 glass-el class 和自訂 background 屬性時，自訂的會蓋掉玻璃背景擇一：只用 glass-el，或移除 glass-el 改自訂
